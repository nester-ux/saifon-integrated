📖 Supercat Automation - User Manual

Welcome to Supercat Automation! Your ultimate tool for automating repetitive tasks. Whether you are managing data entry, automating gameplay, or optimizing your daily workflow, SupercatAuto is designed to be powerful, intelligent, and undetectable.

1. Getting Started (Basic Controls)
You can control SupercatAuto using the main interface or global hotkeys (which can be customized in the Settings tab).

     o Add Point (F8): Captures your current mouse position and adds a new action row.

     o Start (F6) / Stop (F7): Control the main loop.

     o Step Run (Shift+F6): "The Master's Testing Tool" – Executes only the selected row. Essential for debugging complex
       workflows!

2. Understanding the Action Grid The main table is where you build your automation script. 

     0 Type & Action: Defines what the bot will do (e.g., Mouse -> Left Click).

     o X / Y: The screen coordinates for the action.

     o Delay (ms): The wait time after the action is executed (1000 ms = 1 second).

     o Repeat Row: How many times to repeat this specific row before moving to the next.

     o Jump: Enter a row number here to create "Logic Branching." If a condition (Image/Pixel) is met or a task finishes, the bot will
      "Warp" to that specific row.

3. Action Types Explained SupercatAuto offers a wide range of actions to handle any scenario:

     🖱️ Mouse: Supports Left, Right, Middle, or Double clicks. You can also simulate Mouse Wheel scrolling or Drag & Drop actions 
          for complex professional software.

     ⌨️ Key Stroke / Custom Key: Simulate keyboard presses (Enter, Tab, Esc) or custom shortcut combinations (e.g., Ctrl+C, 
          Alt+Tab).

     📝 Text Input:
                o Direct Type: Fast text entry into any active field.

                o Load Text: Syncs with the Dataset tab for mass data entry.

     🎨 Pixel / Screen Sensing

               o Wait for Window: Pauses the script until your target application is detected and active at the front.

               o Smart Pixel Sensor: High-precision sensing that watches for specific UI changes, status bars, or button state transitions.

     👁️ Image Search (Smart Vision)

               o Built-in Snipping Tool: Capture any UI element directly from your screen as a visual reference.

               o Find & Click: Automatically searches for your target icon or button and clicks it instantly.

               o Smart Image Sensor: Holds the script until a specific visual element (like a "Process Complete" notification) appears.

               o Logic Jump: Both Pixel and Image sensors can use the "Jump" feature to skip to a specific row, allowing for complex 
                   branching logic (If-Then-Else)

     ⚙️ System / Utility: Show popup messages, capture screenshots, run external programs (.exe/.bat), or automatically shut down 
           your PC when the task is finished.

4. 🚀 Pro Edition Exclusive Features (Advanced Workflow Optimization) To ensure maximum compatibility with complex
           applications and provide a smooth, error-free experience, SupercatAuto Pro unlocks these intelligent features:

     o Human-like Movement (Bezier Curve): Simulates natural, curved mouse trajectories to interact flawlessly with advanced 
        applications.

     o Dynamic Coordinates (Random X,Y): Prevents repetitive strain on specific UI elements by adding micro-adjustments to click 
        positions.

     o Smart Delay Variance: Introduces natural pauses and slight variations in wait times for maximum workflow stability.

     o Custom Workspace Title: Personalize the application's window name to seamlessly blend into your specific work environment, 
         desktop organization, or screen recording setup.

     o Save & Load Scripts: Export your complex automation workflows as CSV files and reload them anytime for maximum 
        productivity.

5. Using Datasets (CSV Integration) Need to fill out 100 forms with different data?

     1. Go to theDataset tab.

     2. Click Import CSV or paste your data directly into the text box.

     3. In your Action Grid, set the Action to Text Input -> Load Text.

     4. Set the CSV Column # to the column you want to type (e.g., 0 for the first column, 1 for the second). SuperCat will 
         automatically type the data and move to the next line on each loop!

SuperCat Auto Clicker - User Manual

1. Introduction
SuperCat Auto Clicker is a high-performance automation utility designed for Windows. It allows users to record, configure, and execute automated mouse click sequences with precision. The application supports multiple click types, customizable delays, and hotkey integration to streamline repetitive tasks.

2. Getting Started: Adding Click Points

The application operates based on a sequence of coordinates defined in the main data grid. There are two primary methods to capture coordinates:

Visual Selection (Snipping Mode): Click the "Add Point" button. The main interface will hide, and a transparent overlay will appear. Click anywhere on your screen to capture the X and Y coordinates. The application will automatically restore and add the new point to the sequence.

Direct Capture (Hotkey): Position your cursor over the target area and press F8 (default). The coordinates will be recorded instantly without interrupting your workflow.

3. Configuration Grid

Column               Description

1. No.                       The execution order of the action.

2. Description        A customizable label to identify the
                                  specific task (e.g., "Open Folder").

3. X / Y                     The screen coordinates for the click 
                                  action.

4. Delay (ms)          The waiting period (in milliseconds)
                                  before the next action is executed.

5. Action                 The type of mouse event: L-Click, L- 
                                  Double, R-Click, R-Double, M-Click, or 
                                  M-Double.

4. Execution Controls

1. Start Automation: Click the "START" button or press F9. If "Minimize on Start" is checked, the application will move to the background to avoid interfering with the target window.

2. Stop Automation: Click the "STOP" button or press F10 for an emergency halt. The application will immediately cease all activities and restore the main window.

Loop vs. Repeat:
3. Loop: Continuously executes the entire sequence until manually stopped.
4. Repeat: Executes the sequence for a specific number of cycles as defined in the "Repeat" field.

5. File Management

T​o ensure efficiency, users can save and load complex configurations:

1. Save: Export the current coordinate grid and settings into a .csv file.

2. Load: Import previously saved .csv configurations to restore specific automation sequences.

3. Clear All: Resets the grid and removes all current data points.

6. Hotkey Customization

Access the "HotKeys" menu to modify the default triggers. The following actions can be bound to your preferred keys:

1. Add Point: Default is F8.

2. Start: Default is F9.

3. Stop: Default is F10.

7. Licensing and Editions

Free Edition: Supports a maximum of 5 coordinate rows.

Pro Edition: Removes all row limitations, allowing for 
unlimited automation sequences and advanced task management. License status can be verified in the "About" section.
